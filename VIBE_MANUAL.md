# 📘 Vibe Coding & Ralph Loop 사용 설명서 (Seniors Only)

이 프로젝트는 자율형 AI 서브 에이전트인 **랄프(Ralph)**와 함께하는 **바이브 코딩** 환경이 세팅되어 있습니다. 랄프 루프는 당신이 자는 동안에도 코드를 작성하고, 테스트하고, 개선합니다.

## 🛠️ 핵심 구성 요소
1. **`PRD_VIBE.md`**: 랄프의 뇌입니다. 여기에 하고 싶은 말을 적으세요. 랄프는 이 문서를 읽고 작업을 결정합니다.
2. **`LIVE_LOG.md`**: 랄프의 눈입니다. 현재 어떤 작업을 하고 있는지, 어디서 막혔는지 실시간으로 확인할 수 있습니다.
3. **`scripts/ralph-loop.js`**: 랄프의 심장입니다. 루프를 돌리며 실제 세상을 바꿉니다.

## 🚀 시작하기
터미널에서 다음 명령어를 입력하여 랄프 루프를 가동하세요:
```bash
npm run vibe:start
```

## 💡 바이브 코딩 팁 (시니어 개발자 가이드)
- **작은 단위로 지시하세요**: `PRD_VIBE.md`에 너무 큰 작업을 한꺼번에 적지 마세요. 하나씩 해결해 나가는 것이 랄프 루프의 핵심입니다.
- **Git을 신뢰하세요**: 랄프는 모든 성공적인 루프 끝에 커밋을 생성합니다. 마음에 들지 않으면 언제든 `git reset`으로 되돌릴 수 있습니다.
- **에러를 즐기세요**: 랄프는 에러를 만나면 더 강해집니다. `LIVE_LOG.md`에 기록된 에러 로그를 보고 랄프가 스스로 고칠 때까지 지켜보세요.

## 🔒 환경 변수 설정
더 강력한 자율 코딩을 원하신다면 `.env.local` 파일에 다음 항목을 추가하세요:
```env
OPENAI_API_KEY=your_key_here
ANTHROPIC_API_KEY=your_key_here
```
